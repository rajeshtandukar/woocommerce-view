Front-end Product listing of woocommerce can be beautifully listed on list view, date view, grid view and map view with sorting options.

== Description ==
Looking upon the need of users to properly view product listing as per map,list,date,grid this plugin is designed.
This plugin is integreated with Map View meta-box which enables the back-end administrator to add locations to the product items.
Locations can be simply added by drang and drop of the marker or by adding location address or by using current geo-location.

== Installation ==

= Minimum Requirements =

* WordPress 3.5 or greater
* PHP version 5.2.4 or greater
* MySQL version 5.0 or greater



1. Unzip  `wocommerce-view` folder and upload  `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
